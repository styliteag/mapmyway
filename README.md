# MapMyWay — public site

Public GitHub Pages site for the **MapMyWay** iPhone app. Hosts the privacy
policy and support page referenced from the App Store listing and from inside
the app.

- Privacy policy: `index.md` → <https://styliteag.github.io/mapmyway/>
- Support: `support.md` → <https://styliteag.github.io/mapmyway/support>

This repository contains **only** the public-facing site. The application source
lives in the private repository. Do not add app code, internal docs, ADRs, or
review notes here.

Built with GitHub Pages (Jekyll, Cayman theme). Edit the Markdown files and push
to `main`; Pages rebuilds automatically.
